Welcome to CSSS 508: Introduction to R for Social Scientists

Canvas page for course (enrolled students only)

  1. Slides for week 1: Course logistics, R/RStudio, making R Markdown documents
    • Get R
    • Get RStudio
    • RMarkdown documentation
      • HTML document options (global formatting, etc.)
      • PDF document options (requires LaTeX installation to output PDFs)

    • Shapes and line types and color names (PDF) in base R

    • Hadley Wickham on the grammar of graphics
    • Tufte in R (if that's your sort of thing)

    • A cautionary tale about Excel
    • Introduction to dplyr
    • Window functions in dplyr
    • Joining data in dplyr
    • nycflights13 documentation
    • More advanced joins: sqldf for easy SQL in R
  4. Slides for week 4: R data structures
    • Setting up swirl for practice
    • Homework 4: R Markdown template (you will fill this in and submit on Canvas)
